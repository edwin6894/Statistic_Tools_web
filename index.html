Statistic_Tools Version 1 is an Educational/Research Statistical Utility software developed by Edwin Thomas using C#.

This tool offers the support to perform almost every statistical operation within a single platform and facilitates the user with user friendly GUI that helps anyone with basic computer knowledge to operate the software.

The Major purpose of this software is to help High school/Degree/statistical researchers to easily perform any statistical operations that they can use in their projects or research reports.

User Guide

Upon Launching the software the user can see 3 Tabs at the menu bar:

  • OPERATIONS : Provides a sub-menu that contains all Statistical operations
  • THEME : Allow the user to switch themes between dark/white
  • ABOUT : A section that shows the Copyrights of the software and also About the creator

How to Use:

  • Choose your desired operations from the OPERATIONS tab enter the required details in the textfield / the field provided to enter the values and press ENTER button.
  • Upon typing in all values at once and pressing the ENTER key, a new CALCULATE Button appears , Click it to get your output answers.
  • Press the RESET Button to reset the software, and you can choose the next operation to perform
  • For Operations like GEOMETRIC MEAN (FREQUENCY DISTRIBUTION) the data will be displayed in a tabular format on the right-side of the software, a built-in EXPORT Button is provided that exports the data inside the table into .CSV File that the user can save in their PC.
